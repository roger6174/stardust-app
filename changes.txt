[2026-04-04] ASSET MANAGEMENT SYSTEM REDESIGN (Everplans-Inspired)
================================================================================

--- DATABASE CHANGES ---

1. ALTER TABLE `assets`: Changed `category` column from ENUM to VARCHAR(50)
   - Old ENUM: 'Credit Card','Bank Account','Insurance','Property','Legal Document','Nominee','Password','Investment','Vehicle','Collectible'
   - New: VARCHAR(50) — allows any category key string

2. Normalized all category values to lowercase snake_case:
   - 'Credit Card'  → 'cards'
   - 'Bank Account'  → 'banking'
   - 'Insurance' / 'INSURANCE'  → 'insurance'
   - 'Property'  → 'real_estate'
   - 'Legal Document' / 'LEGAL'  → 'legal_documents'
   - 'Nominee'  → 'contacts'
   - 'Password' / 'PASSWORD'  → 'passwords'
   - 'Investment'  → 'investments'
   - 'Vehicle'  → 'vehicles'
   - 'Collectible'  → 'collectibles'
   - 'OTHERS'  → 'others'

3. NEW TABLE `categories`: Reference table for valid categories
   - category_key (PK), label, icon, sort_order, is_active

--- BACKEND API CHANGES ---

4. NEW unified API: /api/vault/:category
   - GET    /api/vault/schemas         → Returns all category schemas (for frontend form generation)
   - GET    /api/vault/summary         → Returns item counts per category
   - GET    /api/vault/search?q=       → Cross-category search
   - GET    /api/vault/:category       → List items in category
   - GET    /api/vault/:category/:id   → Get single item
   - POST   /api/vault/:category       → Add item (schema-validated)
   - PUT    /api/vault/:category/:id   → Update item
   - DELETE /api/vault/:category/:id   → Delete item

5. NEW file: backend/src/config/category_schemas.js
   - Central schema registry (11 categories with field definitions)
   - Used by both backend (validation) and frontend (form generation)

6. NEW file: backend/src/controllers/vaultController.js
   - Replaces: assetController.js, insuranceController.js, passwordController.js,
     legalController.js, contactController.js, othersController.js

7. NEW file: backend/src/routes/vaultRoutes.js
   - Replaces: assetRoutes.js, insuranceRoutes.js, passwordRoutes.js,
     legalRoutes.js, contactRoutes.js, othersRoutes.js

8. MODIFIED: backend/src/app.js
   - Added: app.use('/api/vault', require('./routes/vaultRoutes'))
   - Old routes kept temporarily with deprecation warnings
